# cpp-opencv-workspace

講義用のC/C++と最小構成のopencvが使える開発環境を提供するDocker image

vscodeのremote-containerとして利用されることを想定しています


## 使い方
vscodeの拡張機能であるremote-containerを利用して開くだけです


## 環境

* vscode
* docker


## 注意事項
- vscodeのremote-containerの利用を想定しています
- vscodeなしでも使えます
